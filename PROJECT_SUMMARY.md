## 🔧 Customization

### Easy Swaps

**1. Change Colors:**
Edit `css/styles.css` - Update `:root` variables

**2. Add Real Products:**
Edit `js/data.js` or import from database via API

**3. Update Locations:**
Edit `js/data.js` with actual addresses/phone numbers

**4. Change Text:**
Edit `index.html` - All text is directly in HTML

**5. Update Images:**
Replace image URLs in `js/data.js`

## 🆘 Need Help?

### Common Questions Answered

**Q: Can I use this without a database?**
A: Yes! Remove the API calls and use the sample data in `js/data.js`

**Q: How do I add more products?**
A: Edit `js/data.js` or insert into the `products` table via phpMyAdmin

**Q: Can I change the colors?**
A: Yes! Edit the CSS variables in `css/styles.css`

**Q: Is it mobile friendly?**
A: Yes! Fully responsive from 320px to 2560px

**Q: Can I use it for a real business?**
A: Absolutely! It's production-ready (add SSL & backups)
